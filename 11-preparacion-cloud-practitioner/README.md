# Preparación AWS Certified Cloud Practitioner

## Objetivo

Preparar a los estudiantes para superar la certificación **AWS Certified Cloud Practitioner**.

## Dominios del examen

### Cloud Concepts
Conceptos básicos de cloud computing y ventajas del modelo cloud.

### Security and Compliance
Modelo de responsabilidad compartida, IAM, seguridad y cumplimiento.

### Technology and Services
Servicios principales de AWS y sus casos de uso.

### Billing and Pricing
Modelos de precios, facturación y optimización de costes.

## Estrategia de estudio

1. Comprender los conceptos clave
2. Estudiar los servicios principales
3. Practicar preguntas tipo examen
4. Repasar errores comunes
5. Realizar simulacros completos

## Servicios AWS importantes para el examen

- EC2
- S3
- RDS
- DynamoDB
- VPC
- IAM
- CloudWatch
- CloudTrail
- Route53
- Lambda

## Recomendaciones para el examen

- Leer bien cada pregunta
- Identificar palabras clave
- Eliminar respuestas incorrectas
- Pensar en el servicio más adecuado para cada caso

## Resultado esperado

Al finalizar el curso, el estudiante debería ser capaz de:

- Explicar los fundamentos de la nube
- Identificar servicios principales de AWS
- Comprender el modelo de responsabilidad compartida
- Entender conceptos básicos de seguridad y costes en AWS
