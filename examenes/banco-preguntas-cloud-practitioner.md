# Banco inicial de preguntas Cloud Practitioner

## Pregunta 1

**¿Qué característica de la nube permite aumentar o reducir recursos automáticamente según la demanda?**

A. Alta disponibilidad  
B. Elasticidad  
C. Gobernanza  
D. Cumplimiento  

**Respuesta correcta:** B

**Explicación:**  
La elasticidad permite adaptar recursos a la demanda real. Alta disponibilidad busca mantener el servicio operativo, pero no implica por sí sola ajuste dinámico de capacidad.

## Pregunta 2

**¿Qué servicio de AWS se utiliza principalmente para almacenamiento de objetos?**

A. Amazon EBS  
B. Amazon EC2  
C. Amazon S3  
D. Amazon RDS  

**Respuesta correcta:** C

**Explicación:**  
Amazon S3 es el servicio de almacenamiento de objetos de AWS. EBS es almacenamiento en bloque, EC2 es cómputo y RDS es base de datos gestionada.

---

## Pregunta 3

**¿Quién es responsable de la configuración de usuarios y permisos en una cuenta AWS?**

A. AWS  
B. El cliente  
C. El proveedor de internet  
D. El centro de datos físico  

**Respuesta correcta:** B

**Explicación:**  
Según el modelo de responsabilidad compartida, AWS protege la infraestructura, pero el cliente gestiona accesos, identidades y permisos.

---

## Pregunta 4

**¿Qué servicio permite ejecutar máquinas virtuales en AWS?**

A. AWS Lambda  
B. Amazon S3  
C. Amazon EC2  
D. Amazon DynamoDB  

**Respuesta correcta:** C

**Explicación:**  
Amazon EC2 proporciona capacidad de cómputo elástica mediante instancias virtuales.

## Pregunta 5

**¿Qué servicio se usa para gestionar identidades y permisos en AWS?**

A. Amazon VPC  
B. AWS IAM  
C. Amazon CloudFront  
D. AWS Config  

**Respuesta correcta:** B

**Explicación:**  
AWS IAM permite crear usuarios, grupos, roles y políticas de permisos.

## Pregunta 6

**¿Qué servicio de AWS ayuda a registrar llamadas a la API con fines de auditoría?**

A. Amazon CloudWatch  
B. AWS CloudTrail  
C. Amazon Inspector  
D. AWS Shield  

**Respuesta correcta:** B

**Explicación:**  
CloudTrail registra actividad de API y eventos de cuenta. CloudWatch se orienta más a métricas, logs y alarmas.

## Pregunta 7

**¿Cuál es una ventaja principal del modelo de pago por uso en cloud?**

A. Obliga a comprar hardware por adelantado  
B. Elimina toda configuración de seguridad  
C. Permite pagar solo por los recursos consumidos  
D. Impide escalar servicios  

**Respuesta correcta:** C

Una ventaja clave del cloud es el consumo bajo demanda y el pago por uso real.

## Pregunta 8

**¿Qué servicio de base de datos NoSQL totalmente gestionado ofrece AWS?**

A. Amazon Aurora  
B. Amazon RDS  
C. Amazon DynamoDB  
D. Amazon Redshift  

**Respuesta correcta:** C

**Explicación:**  
DynamoDB es NoSQL. RDS y Aurora son relacionales; Redshift es analítica/data warehouse.
