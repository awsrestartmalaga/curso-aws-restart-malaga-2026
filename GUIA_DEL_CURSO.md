# Guía del curso AWS re/Start Málaga 2026

## Bienvenida

Este repositorio sirve como base de estudio y apoyo para el curso AWS re/Start Málaga 2026.

El objetivo es que el alumno pueda:

- seguir los contenidos del curso de forma ordenada
- reforzar conceptos técnicos
- practicar preguntas tipo examen
- preparar la certificación AWS Certified Cloud Practitioner
- construir una base útil para empleabilidad

## Cómo estudiar bien este repositorio

1. Leer el README del bloque
2. Estudiar el tema
3. Anotar conceptos clave
4. Practicar preguntas
5. Hacer laboratorios
6. Repasar errores frecuentes

## Qué debe conseguir el alumno

Al terminar el curso, el alumno debería ser capaz de:

- explicar qué es cloud computing
- identificar servicios principales de AWS
- entender fundamentos de seguridad, red, cómputo y almacenamiento
- manejar vocabulario técnico básico
- razonar preguntas tipo Cloud Practitioner

## Recomendación docente

No estudiar solo memorizando servicios.
Primero entender:

- problema
- necesidad
- servicio adecuado
- coste
- seguridad
- operación básica
