# Qué es Cloud Computing

## Definición

Cloud computing consiste en utilizar recursos informáticos a través de internet bajo demanda, sin necesidad de comprar o mantener infraestructura física.

Los recursos pueden incluir:

- servidores
- almacenamiento
- redes
- bases de datos
- servicios de inteligencia artificial

## Diferencia con el modelo tradicional

### Infraestructura tradicional (On-Premise)

Una empresa debe:

- comprar servidores
- mantener hardware
- gestionar el datacenter
- actualizar equipos
- pagar electricidad y refrigeración

### Modelo Cloud

En la nube:

- los servidores se crean en minutos
- no hay inversión inicial
- se paga solo por uso
- la infraestructura la gestiona el proveedor cloud

## Ejemplo sencillo

Una startup quiere lanzar una web.

**On-premise**

- compra servidores
- instala hardware
- configura red
- mantiene infraestructura

**Cloud**

- crea una instancia EC2
- sube la aplicación
- paga solo mientras esté encendida

## Ventajas principales

- escalabilidad
- elasticidad
- pago por uso
- rapidez de despliegue
- alta disponibilidad
