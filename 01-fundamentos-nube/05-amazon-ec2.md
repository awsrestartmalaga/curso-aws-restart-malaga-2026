# Amazon EC2

## QuÈ es EC2

Amazon EC2 permite ejecutar **servidores virtuales en la nube**.

Estos servidores se llaman **instancias**.

## Para quÈ se usa

EC2 se utiliza para:

- alojar aplicaciones web
- ejecutar software empresarial
- servidores de desarrollo
- servidores de pruebas

## Ventajas

- escalabilidad
- control del sistema operativo
- pago por uso

## QuÈ aprender· el alumno

- lanzar una instancia EC2
- conectarse al servidor
- detener la instancia
- eliminar la instancia
