# Modelo de responsabilidad compartida

En AWS la seguridad es una responsabilidad compartida entre AWS y el cliente.

## AWS es responsable de

La **seguridad de la nube**.

Incluye:

- centros de datos
- hardware
- red global
- infraestructura física

## El cliente es responsable de

La **seguridad en la nube**.

Incluye:

- usuarios y permisos
- configuración de servicios
- sistemas operativos
- aplicaciones
- datos

## Ejemplo

Si usas EC2:

AWS protege:

- datacenter
- servidores físicos
- red

El cliente protege:

- sistema operativo
- firewall
- usuarios
- datos
