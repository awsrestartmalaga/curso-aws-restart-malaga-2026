# Categorías de servicios AWS

AWS ofrece más de 200 servicios.

Para entenderlos mejor se agrupan en categorías.

## Compute

Servicios que permiten ejecutar aplicaciones.

Ejemplos:

- Amazon EC2
- AWS Lambda
- Elastic Beanstalk
- ECS

---

## Storage

Servicios para almacenar datos.

Ejemplos:

- Amazon S3
- Amazon EBS
- Amazon EFS
- Glacier

---

## Networking

Servicios para conectividad de red.

Ejemplos:

- Amazon VPC
- Route 53
- Elastic Load Balancer
- CloudFront

---

## Databases

Servicios de bases de datos gestionadas.

Ejemplos:

- Amazon RDS
- Amazon DynamoDB
- Amazon Aurora
- Amazon Redshift

---

## Security

Servicios para seguridad y control de acceso.

Ejemplos:

- AWS IAM
- AWS Shield
- AWS WAF
