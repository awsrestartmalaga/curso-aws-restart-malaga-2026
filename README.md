# AWS re/Start Málaga 2026

Repositorio de apoyo para el curso **AWS re/Start Málaga 2026**, orientado a reforzar conocimientos técnicos, preparación de la certificación **AWS Certified Cloud Practitioner** y empleabilidad.

## Objetivos del repositorio

- Reunir materiales organizados por bloques temáticos.
- Facilitar el estudio progresivo del curso.
- Aportar resúmenes, glosarios y preguntas tipo examen.
- Centralizar laboratorios, ejercicios y pequeńos proyectos.
- Ayudar al alumnado a preparar entrevistas y construir portfolio.

## Cómo usar este repositorio

1. Leer el README de cada bloque
2. Estudiar los conceptos principales
3. Realizar ejercicios y labs
4. Practicar preguntas tipo examen
5. Repasar antes de cada módulo

## Objetivo del curso

Preparar a los estudiantes para:

- Comprender los fundamentos de Cloud
- Trabajar con servicios básicos de AWS
- Preparar la certificación **AWS Certified Cloud Practitioner**
- Desarrollar habilidades técnicas iniciales para trabajar en Cloud

## Certificación objetivo

AWS Certified Cloud Practitioner

Dominios principales:

- Cloud Concepts
- Security and Compliance
- Technology and Services
- Billing and Pricing
